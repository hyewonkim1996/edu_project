## 📋 프로젝트 개요 및 목표
> 개발 동기 : 디지털 소외 계층의 디지털 교육 격차 심화

## ❓ 주요 사용 기술과 이유

|사용 기술|프로젝트 적용 범위|🌟이유🌟|
|------|---|---|
|전자정부 표준프레임워크|프로젝트 개발환경, DB 연동 등 설정, 서비스단, DAO단|공통컴포넌트와 개발 표준화로 공공 서비스 부문 통일성 강화, 유지보수 용이|
|Spring Security|회원가입, 로그인, 로그아웃|비밀번호 암호화 및 검증된 로그인 절차 사용으로 회원정보 보안 강화|
|REST API|카카오 로그인, 회원가입, 댓글|코드 가독성 향상|
|Session|비회원 사용 기능 제한, 로그아웃, 수강후기|잦은 DB 조회보다 서버에 회원 정보 등록해 보안 강화 및 불필요한 중복 READ 코드 생략|
|DB - View|수강 후기 작성 시 수강 강좌 카테고리에 회원 수강 강좌 반영|기존 후기+강좌 조인한 4줄 쿼리->1줄로 줄여 가독성, 편의성 향상 및 속도 개선|
|open API|카카오 로그인|SNS 간편 로그인으로 사용자의 편의성 향상 및 더 많은 회원 유치 가능|
|lombok 라이브러리|각 VO @Getter, @Setter, @toString|코드 다이어트로 속도 개선|
